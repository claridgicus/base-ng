# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in Base NG, please report it responsibly.

For vulnerabilities related to the original Base UI components, please report to the upstream project at [security@mui.com](mailto:security@mui.com).

For Angular-specific issues in this port, please open a private security advisory on this repository.

We appreciate your help in keeping Base NG secure.
